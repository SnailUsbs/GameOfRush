# GameOfRush
Game Of Life mod for Bomb Rush Cyberfunk

![](https://github.com/SnailUsbs/GameOfRush/blob/main/Images/InGameOfRush.png)

This is a simple mod that adds in conway's game of life into Bomb rush Cyberfunk

# Features:

- **Can be fully used from the in-game phone**

- **Game Will Auto Restart Once It Hits A Dead End**

- **Adjustable Game Speed/Colors**

- **Option To Have The Game Run In The Background While The In-Game Phone Is Closed**

# Important Info:

- Since this is more of a fun proof of concept rather than a truly useful or fun, It will not be going up on thunderstore, and instead will only be held in this github repo.
